Get Loquivox

Pick your flavor. Every path ends with you talking to your computer.

# with an AUR helper yay -S loquivox # allow global hotkeys, then log out & back in sudo usermod -aG input $USER

Builds a static whisper-cli for offline transcription out of the box.

# download the latest .deb from the releases page, then: sudo apt install ./loquivox_*_amd64.deb # allow global hotkeys, then log out & back in sudo usermod -aG input $USER

Get the package on the releases page.

# clone and run the setup script — it detects your distro & session git clone https://github.com/Xhelliom/loquivox.git && cd loquivox ./setup.sh # allow global hotkeys, then log out & back in sudo usermod -aG input $USER # launch loquivox

Works on Debian, Ubuntu and Arch — X11 and Wayland are auto-detected.
